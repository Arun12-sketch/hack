# Swap 페이지 가격 차트 - v0.7.0

## 문제 정의

### 현상
- `/swap` 페이지에 스왑 카드만 존재하며 (max-w 480px, 중앙 정렬), 토큰 쌍의 가격 추이 정보가 없음
- 사용자가 현재 가격만 보고 스왑해야 하며, 과거 가격 변동을 확인할 수 없음
- 경쟁 DEX(Hybra Finance 등)는 스왑 옆에 가격 차트를 제공하여 더 나은 트레이딩 경험을 제공

### 원인
- 초기 구현에서 스왑 기능에만 집중, 가격 시각화 기능 미구현
- 온체인 가격 히스토리 데이터 수집 인프라 미구축 (현재 단계에서는 mock 데이터로 대체)

### 영향
- 사용자가 가격 트렌드를 파악하지 못해 불리한 타이밍에 스왑할 수 있음
- DEX 프론트엔드의 완성도와 신뢰감 부족
- 트레이딩 의사결정에 필요한 핵심 정보 부재

### 목표
1. `/swap` 페이지를 2컬럼 레이아웃으로 변경 (왼쪽: 스왑, 오른쪽: 가격 차트)
2. Recharts 기반 Area Chart로 토큰 쌍의 가격 추이 표시
3. Mock 데이터를 사용하여 ~1개월 기간의 가격 데이터 시각화
4. 선택된 토큰 쌍에 따라 차트가 동적으로 변경
5. 반응형 레이아웃 (모바일: 세로 스택, 데스크탑: 가로 배치)

### 비목표 (Out of Scope)
- 실제 온체인/오프체인 가격 데이터 연동 (mock 데이터만 사용)
- 시간 범위 선택 버튼 (1H, 1D, 1W 등) — 고정 기간만 표시
- 캔들스틱/OHLC 차트 — Area Chart만 구현
- Dust Token Exchange 등 추가 기능
- 주문 히스토리, 오더북

## 제약사항
- **차트 라이브러리**: Recharts (사용자 선택)
- **데이터**: mock 데이터로 구현, 추후 실제 데이터 연동 가능한 구조
- **기존 스왑 기능**: 현재 스왑 로직/UI 변경 최소화 (레이아웃 변경만)
- **디자인 참조**: Hybra Finance trade 페이지 (2컬럼, Area Chart with gradient fill)
