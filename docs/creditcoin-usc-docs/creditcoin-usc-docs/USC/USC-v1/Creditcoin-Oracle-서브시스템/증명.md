# 증명 (Attestation)

> **참고:** 이 문서는 더 이상 사용되지 않는 버전입니다. 최신 [[USC/소개|USC 문서]]를 참조하세요.

## 개요

**증명(Attestation)**은 Creditcoin 탈중앙화 오라클이 외부 소스 체인의 완결된 상태를 추적하는 프로세스입니다.

이 시스템은 단일 엔티티를 신뢰하는 대신, 블록체인 상태 전환을 모니터링하는 여러 독립적인 증명자(attestors)에 의존합니다.

## 증명 프로세스

1. 증명자들은 새로 완결된 블록을 위해 소스 체인을 지속적으로 관찰합니다.

2. 증명자들은 [검증 가능한 무작위 함수(VRF)](https://en.wikipedia.org/wiki/Verifiable_random_function)와 같은 메커니즘을 사용하여 특정 블록 높이에서 증명할 자격이 있는지 확인합니다.

3. 자격이 있는 증명자들은 소스 체인 진행 상황의 요약을 생성하고 서명된 증명을 투표로 제출합니다.

4. Creditcoin 검증자들은 투표 자격을 확인하고 동일한 다이제스트 값과 일치하는 유효한 증명을 축적합니다.

5. 투표가 필요한 임계값을 초과하면 증명이 확인되고 온체인에 저장됩니다.
