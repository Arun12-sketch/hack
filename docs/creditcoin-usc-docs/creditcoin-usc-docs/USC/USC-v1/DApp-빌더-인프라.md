- [[USC/USC-v1/DApp-빌더-인프라/소스-체인-스마트-컨트랙트|소스 체인 스마트 컨트랙트]]: 소스 체인 스마트 컨트랙트에 대한 개요와 Creditcoin Oracle을 사용하여 Creditcoin에서 해당 데이터를 안전하게 쿼리하기 위한 모범 사례 패턴입니다.
- [[USC/USC-v1/DApp-빌더-인프라/유니버설-스마트-컨트랙트|유니버설 스마트 컨트랙트]]
- [[USC/USC-v1/DApp-빌더-인프라/DApp-디자인-패턴|DApp 디자인 패턴]]: Universal Smart Contract로 구동되는 DApp에서 원활한 사용자 상호작용을 가능하게 하는 일반적인 디자인 패턴에 대한 개요입니다.

- [[USC/USC-v1/DApp-빌더-인프라/USC-튜토리얼|USC 튜토리얼]]: Creditcoin USC를 배우기 위한 비디오 튜토리얼입니다.
